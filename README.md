# MechList

Mod used for whitelisting on mechanist server.

## Config

You need to get the api key from google api. The file will be generated under /config/mechlist/config.json

```json
{
  "spreadsheetId": "SHEET-ID",
  "sheetRange": "RANGE",
  "shouldOpAll": false,
  "api-key": "API-KEY"
}
```
